Projet réalisé dans le cours IMA201 de Télécom Paristech.

Il s'agit d'un travail réalisé à partir de l'article: 
Image Fusion with Guided Filtering Shutao Li, Member, IEEE, Xudong Kang, Student Member, IEEE, and Jianwen Hu. 
Le but était de créer à partir de plusieurs images d'un même objet mais avec différentes mises au point, une image avec une profondeur de champ élargie.
